CSC1034: Practical 1
====================

Portfolio 1
===========

This package is build as a part of the CSC1034: Portfolio-1.

Type `python walking_panda.py` in terminal to run the program.

**Any exceptions are likely to be caused by pandawalk.ogg - make sure to read --soundpath argument.**

Arguments (type after python walking_panda.py)

>--scale "float": changes size of panda

>--no-rotate: stops camera from rotating

>--no-move: stops panda from moving

>--colour "string = 'red', 'green', 'blue', 'yellow', 'purple', 'orange'": changes colour of panda based on RGB values

>--first-person: makes camera follow panda from panda's perspective

>--soundpath: use this to input local file path for pandawalk.ogg in linux format e.g. "/c/Users/jacob/PycharmProjects/c1007711_csc1034_practical1_2021/walking_panda/pandawalk.ogg"
